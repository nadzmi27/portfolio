// CI/CD flow
"GitHub" > "Cloudflare Worker - CI/CD": push to main triggers build
"Cloudflare Worker - CI/CD" > "Cloudflare Pages - Hosting": build and deploy new version
"Cloudflare Pages - Hosting" > "Client - User's Device": serves app bundle on page load

// Auth flow
"Auth Context" > "Auth": login/signup request, redirects to Google OAuth, returns with session token on success
"Auth" > "Google OAuth": redirects for authentication
"Google OAuth" > "Auth": returns token
"Auth" > "Auth Context": returns session token

// Load list flow
"Services - Supabase Client" > "Postgres DB": fetch user_words_full view, returns words + entries + collections joined
"Postgres DB" > "Services - Supabase Client": return words + entries + collections

// Background audio
"Pages and Components" > "Cloudflare R2": stream background rain audio

// Word add flow
"Pages and Components" > "Services - Supabase Client": add word to collection
"Services - Supabase Client" > "Postgres DB": check fetched_terms cache, returns hit (insert directly) or miss (invoke edge function)
"Postgres DB" > "Services - Supabase Client": cache hit or miss

"Services - Supabase Client" > "Edge Function: add-word-user-collection": invoke on cache miss
"Edge Function: add-word-user-collection" > "Auth": verify bearer token, get user id
"Edge Function: add-word-user-collection" > "Merriam-Webster APIs": fetch collegiate entry and synonyms/antonyms in parallel
"Merriam-Webster APIs" > "Edge Function: add-word-user-collection": collegiate entries (or suggestions if not found) merged with syns/ants by shared uuid

"Edge Function: add-word-user-collection" > "Postgres DB": save the word, remember it was fetched, and add it to the user's collection
"Postgres DB" > "Edge Function: add-word-user-collection": entries by headword, ordered by homograph
"Edge Function: add-word-user-collection" > "Services - Supabase Client": word + entries
